
# Images du Calendrier de l'Avent

Veuillez placer vos photos de chat dans ce dossier.
Les fichiers doivent être nommés exactement comme suit :

- card_1.jpg
- card_2.jpg
- card_3.jpg
...
- card_24.jpg

Assurez-vous que les images sont au format JPG.
Si vous utilisez un autre format (PNG, etc.), vous devrez mettre à jour le code dans `components/advent/adventData.ts`.
